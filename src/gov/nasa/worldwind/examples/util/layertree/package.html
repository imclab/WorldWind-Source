Displays a list of layers using {@link gov.nasa.worldwind.util.tree.BasicTree}. The layer tree also supports displaying a tree of KML features.

See {@link gov.nasa.worldwind.examples.LayerTreeUsage} for an example of how to use the layer tree. See {@link gov.nasa.worldwind.examples.kml.KMLViewer} for an example of using the layer tree to display the contents of a KML document.

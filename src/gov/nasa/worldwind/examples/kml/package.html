Examples of importing and exporting files in the Keyhole Markup Language (KML).

{@link gov.nasa.worldwind.examples.kml.KMLViewer} is an example application that allows the user to import a KML file into a WorldWind layer. The contents of the file are displayed in a tree. {@link gov.nasa.worldwind.examples.kml.ExportKML} shows how to export WorldWind shapes to KML format.
